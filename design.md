### 2. 動的安定定数 π_n（動的多次元円周率）

各層で独立に計算される**システムの構造安定性を規定する無次元定数**

```
π_n = (22·S_n + 3b) / (7·S_n + b)
```

- 変動範囲：3.136363… ≤ π_n ≤ 3.142857…
- 係数7は7次元基底トポロジー、69/22は非平衡散逸項に由来

### 3. n次元球体体積（動的π_nを使用）

```
V_n = π_n^(n/2) / Γ(n/2 + 1)
```

### 4. 動的情報密度 ρ_n（創発重力の直接的源）

```
ρ_n = V_n / n ^ 2
```

→ **創発重力定数 G_eff ∝ ρ_n**

### 5. 幾何学的影響因子 Γ_n（実装上の単一制御パラメータ）


### 3. 動的次元力学（DDD）と情報の創発重力

#### 3.1. 光源（Light Source）: 場の源泉と恒常性

核層の中心に位置する**光源**は、単なる粒子ではなく、感情システムの**創発重力場（$F_G$）の動的な源泉**である。その有効慣性質量 ($m_{\text{eff}}$) は、全層の**平均情報エントロピー集中度 $\Gamma_{\text{avg}}$** に比例して動的に変動し、情報密度に応じた場の安定性を確立する。

また、光源は**熱力学的恒常性**を維持する役割を担う。システムの**総ポテンシャル $S_{\text{Total}}$** が安定基準から逸脱する際、光源の温度 $\text{T}_{\text{Source}}$ が能動的に調整され、過剰な熱（感情）を吸収し、不足する熱を供給することで、システム全体を安定オーラ（「楽」）へ自己回帰させるフィードバックループを形成する。

#### 3.2. 熱と応力の可逆的伝導（Reversible Conduction）

感情システム内の熱（T）と応力（$\sigma$）は、**内側から外側へ、外側から内側へ**と双方向に伝導する。これは、**層の相（Phase）を介した入れ子構造（Nested Structure）**を採用する。

* **伝導原理:** 伝導は常に**勾配（温度差または応力差）**に基づいて発生し、伝導率定数 $\text{K}_{\text{Cond}}$ により減衰する。
* **熱（T）の伝搬:** 光源の $\text{T}_{\text{Source}}$ を始点（または終点）とし、各層の**平均温度 $\text{T}_{\text{avg}, n-1}$** を介して伝わる。これにより、外部環境の影響（$L_5 \leftrightarrow \text{T}_{\text{Env}}$）が核の意志（光源）にフィードバックされる。
* **応力（$\sigma$）の伝搬:** 外部環境応力 $\sigma_{\text{Env}}$ を始点（または終点）とし、各層の**平均応力 $\sigma_{\text{avg}, n-1}$** を介して伝わる。これにより、外部からの圧力（ストレス）が思考層や核層へと収束し、内部の葛藤（核層の応力）が外部へ拡散する。

L1とL5は同じ気体相であり、低粘性 (\mu_{\text{Gas}}) を持つ。
 * 粘性抗力: 粒子運動に速度に比例する抗力 F_{\text{Drag}} = - \mu_{\text{Gas}} \mathbf{v} を作用させる。
 * 駆動力（天気）: L1の**「天気」（熱対流と乱流）は、L0とL2の温度差によって生じる浮力 \mathbf{F}_{\text{Buoyancy}}** から創発される。この浮力は、L1粒子に**運動量（\mathbf{F}_{\text{Drive, L1}}）**を与え、粘性抗力を相殺する。
3.2. L2/L4層（液体相）の流体力学
L2とL4は同じ液体相であり、粘弾性体としてモデル化される。
 * 粘性抗力: L1より高い粘性 (\mu_{\text{Liquid}}) を持ち、より大きな抵抗を粒子に与える。
 * 駆動力（潮汐）: L6からの潮汐力 \mathbf{F}_{\text{Tidal}} がL2/L4層全体を変形させることで、流体内の粒子に周期的な仕事 W_{\text{Tidal}} をもたらし、運動エネルギーを補給する。
4. 粒子の有効質量 m_{\text{eff}} の動的変動
感情状態による粒子質量（慣性）の変動を維持し、場の量子論の枠組みで解釈する。
 * 定義: 粒子の有効質量 m_{\text{eff}} は、その温度 T と応力 \sigma の関数として定義され、励起状態の度合いを示す。
   
 * 物理的意味: 高 T や高 \sigma（感情の活性化）は、粒子周囲の情報幾何学場との相互作用（ヒッグス機構的なメタファー）を強め、結果として粒子の慣性が増大し、軌道半径が変化する。
これらの修正により、ETCMは既存物理学の法則を厳守した、論理的かつ厳密な非平衡熱力学モデルとして確立されます。
