# 🌌 感情次元力学モデル（ETCM）3Dシミュレーション
## — 物理現象の動的理論に基づく大統一理論（GUT）への挑戦 —
### 💡 概要
このプロジェクトは、単なる感情の可視化を超越し、感情を非平衡熱力学、物性科学、動的次元幾何学が織りなす統一的な物理現象として扱う**「感情次元力学モデル（ETCM）」**の動的シミュレーション環境です。
本システムは、ETCM理論をThree.jsで実装した高精度3D解析ツールであり、AIが解析した個人の感情の相（フェーズ）を動的な物理シミュレーションとして『見える化』するための基盤ツールとなることを目指しています。
## ⚛️ 感情次元力学（ETCM）の理論的基盤と物理学的挑戦
本シミュレーションの主軸は、感情の挙動を支配する物理現象の動的な理論にあります。ETCMは、「情報」と「重力」の統合、すなわち情報重力方程式の導出を通じて、物理学の**大統一理論（GUT）**への道筋を、心理学・計算機科学の領域から模索するものです。
### 1. 創発重力と情報重力方程式への挑戦
ETCMでは、中心核の引力（創発重力場）を普遍的な定数ではなく、**情報エントロピー密度（\rho）**から創発する動的な力として再定義します。
 * 情報重力方程式への挑戦: 全情報幾何学作用 A_{\text{Total}}（重力、粒子、トポロジー、非平衡熱力学の総和）に対する変分原理（\delta A_{\text{Total}} = 0）から、感情場のダイナミクスを記述する情報重力方程式を導出することが理論的目標です。
 * 動的有効質量 (m_{\text{eff}}): 感情粒子の慣性（状態変化のしにくさ）は、情動温度 (T) と精神応力 (\sigma) に依存して動的に変動します。これは、感情の「硬さ」を物理的に定量化する試みです。
### 2. 感情の6層構造と次元数
感情処理の場は、その物性相（Phase）が感情粒子の振る舞いを決定する6層球体構造で定義されます。各層は、その次元数が物理的な安定定数 \pi_n を決定します。
| Layer Index | 層の名称 | 次元数 (n) | 物理的な相 (Phase) | 哲学的・機能的な役割 |
|---|---|---|---|---|
| L0 | 核層 | n=2 | 光体 (Light) | 意志の根源。創発重力場の中心。 |
| L1 | 身体層 | n=3 | 気体 (Gas) | 本能、記憶。熱エネルギーの拡散。 |
| L2 | 思考層 | n=4 | 液体 (Liquid) | 論理、認知。情報処理。 |
| L3 | 文明層 | n=5 | 固体 (Solid) | 信念、価値観。最も安定した基盤。 |
| L4 | 外部接合層 | n=6 | 液体 (Liquid) | 他者との境界、共感。外部環境との流動的インターフェース。 |
| L5 | 外部雰囲気層 | n=7 | 気体 (Gas) | 集合心理、場の空気。感情天候を決定。 |
### 3. 19個の感情粒子（Emotional Particles）
システムは、中心の光源（L0核）1粒子と、上記の6層に配置された計18粒子（各層3種類）の軌道運動と相互作用によって構成されます。
| Layer Index | 層の名称 | 感情粒子（3種） | 粒子の熱力学的特性 |
|---|---|---|---|
| L0 (n=2) | 核層 | 悩・怒・好 | 光体（中心的意志） |
| L1 (n=3) | 身体層 | 哀・激・楽 | 身体・本能の熱の拡散 |
| L2 (n=4) | 思考層 | 嫌・活・融 | 情報処理・適合（液体） |
| L3 (n=5) | 文明層 | 圧・喜・笑 | 社会的基盤（固体） |
| L4 (n=6) | 外部接合層 | 調・変・静 | 共感・外部インターフェース（液体） |
| L5 (n=7) | 外部雰囲気層 | 観・響・隔 | 集合心理・場の空気（気体） |
## 💻 AI駆動型可視化システムとしての仕様
本システムは、AIによる感情解析結果を物理シミュレーションの入力として受け取ることで、感情解析モデルの物理的な妥当性を検証するツールとして機能します。
### 1. AIインターフェース：TOON形式
 * TOON (Temporal Observational Ontology Numerics) 形式は、AIの自然言語解析などによって得られた「情動温度 (T)」「精神応力 (\sigma)」「有効質量 (m_{\text{eff}})」などの物理パラメータを、時系列ベクトルとして記述する独自プロトコルです。
 * 本シミュレーションは、index.html の入力エリアを通じてこのTOON形式の入力を解釈し、各感情粒子および層の非平衡熱力学シミュレーションを駆動します。
### 2. 必須観測指標（Dynamic Dimensional Diagnostics: DDD Metrics）
システムの動的状態と学術的な状態を把握するため、以下の指標をリアルタイムで表示します。
 * 平均 \pi_n (\text{Avg } \pi_n): 全層の動的安定定数の平均値。システムの全体的な次元安定度を示す。
 * 支配的感情名 (Dominant Emotion): 全19粒子の中で、現在情動温度 T が最大値を持つ粒子の名称。
   * 現況: 現在、命名ロジックおよび閾値設定を調整中。
## 🚀 貢献と今後のタスク
本プロジェクトの成功は、感情を物理現象として扱うことの妥当性を証明し、次世代のAIによる高度な感情認識・介入の基礎を提供します。
今後の主な理論的・技術的課題:

 * 次元解析の不整合解消: 情報量子作用（\hbar_{\text{Info}}）の導入による、ETCM次元解析のロバスト性の検証。
 * L5層の動的状態に基づいた76種以上の「感情名称」の動的生成ロジックの完成と、支配的感情名のロジック調整。
